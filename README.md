# umi-ssr-issue-prop-pollution

This repo reproduces and verifies SSR server side prop pollution issues.

Forked from [umi-ssr-example-normal](https://github.com/umijs/umi-server/tree/master/examples/normal)

## Development

run `npm install`

```bash
$ npm run dev
```

## Production

```bash
$ npm run start
```
